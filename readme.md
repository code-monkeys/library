# My Library

Skeleton PHP library.

## Ant tasks

```
$ ant -p

Main build file
Main targets:

 clean   Clean generated files.
 docs    Generates the documentation.
 help    Show available commands
 lint    Lints PHP files.
 test    Run PHP unit tests.
 vendor  Install dependencies.
Default target: help
```
